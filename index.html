What is BFS (Breadth-First Search)?

Date: 10 February 2026

BFS explores nodes level by level. It starts from a chosen node, visits its closest neighbours first, then explores the neighbours of those nodes. It typically uses a queue to track which node should be explored next.

Intro to normalisation in databases

Date: 10 February 2026

Normalisation organises tables so each table focuses on one topic and duplicated data is minimised. This helps keep data consistent and easier to maintain.
