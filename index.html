What is BFS (Breadth-First-Search)? (short explanation)

Date: 10 February 2026

BFS or Breadth-First-Search is a searching algorithm that explores every node in a given data structure. It selects a starting point and then from that node it explores its closest neighbours and after searching all of its closest neighbours, it explores the neighbours of the nodes it has already visited. In the end it stores all of the nodes it has visited and removes it from the unvisited or the queue.

Intro to normalisation in databases

Date: 10 February 2026

Normalisation is the process of organising tables in a database so that each table focuses on one topic and there is minimal duplicated data. This helps keep the data consistent and easier to maintain.

What is an HTTP request?

When I type a website address into my browser and press Enter, my computer sends an HTTP request to a web server. This request is a structured message that asks the server for a specific resource, such as a web page or an image. The server receives the request, processes it, and then sends back an HTTP response. The response includes a status code, for example 200 OK if everything worked, and the content that should be displayed. My browser reads this response and shows the page on the screen. This request–response cycle happens many times when I visit a modern website, because the page often needs extra files like images, CSS, and JavaScript.
