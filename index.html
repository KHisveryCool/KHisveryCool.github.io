What is BFS (Breadth-First Search)?

Date: 10 February 2026

BFS (Breadth-First Search) is a searching algorithm that explores nodes level by level. It starts from a chosen node, visits its closest neighbours first, then explores the neighbours of those nodes. It typically uses a queue to track which node should be explored next.

Intro to normalisation in databases

Date: 10 February 2026

Normalisation is the process of organising tables in a database so each table focuses on one topic and there is minimal duplicated data. This helps keep the data consistent and easier to maintain.

What is an HTTP request?

When I type a website address into my browser and press Enter, my computer sends an HTTP request to a web server. This request asks the server for a specific resource, such as a web page or an image. The server then sends back an HTTP response with a status code (e.g., 200 OK) and the content to display. This request–response cycle happens many times on modern websites because pages often load extra files like images, CSS, and JavaScript.
